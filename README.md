# sales-revenue-dashboard-PowerBI
Sales and revenue analysis dashboard built in Power BI with insights on trends, performance, and customer behavior.

📊 Sales & Revenue Dashboard

📌 Project Overview

This project focuses on analyzing sales and revenue data using an interactive Power BI dashboard.
The dataset used is from Kaggle (public dataset).

---

🛠️ Tools Used

- Power BI
- Data Cleaning
- Data Visualization

📈 Key Insights

- Electronics category generated the highest revenue, followed closely by Clothing
- Peak revenue observed in May (~53K), with fluctuations across months indicating seasonal trends
- Female customers contributed slightly higher revenue (~51%) compared to male customers
- Customers aged around 40–45 generated the highest revenue
